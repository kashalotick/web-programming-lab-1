# Лабораторна робота 1 — Консольний застосунок "Готель"

## Предметна область
Готель: керування номерами, відвідувачами та бронюваннями.

## Реалізовано
- Maven-проєкт на Java 17.
- Сутності предметної області:
  - `Hotel`
  - `Room`
  - `Visitor`
  - `Booking`
- Бізнес-логіка:
  - бронювання номера з розрахунком загальної суми;
  - оновлення статистики готелю (вільні/заброньовані номери);
  - виведення статистики бронювань;
  - CRUD номерів;
  - CRUD відвідувачів.
- Імпорт/експорт CSV:
  - номери (`rooms.csv`);
  - відвідувачі (`visitors.csv`);
  - підтримка сортування перед експортом.
- Консольне меню для демонстрації функціональностей.

## Структура
- `model` — доменні класи.
- `service` — бізнес-логіка та CRUD-операції.
- `io` — імпорт/експорт CSV.
- `ui` — консольне меню.

## Тести
Додано більше ніж 9 модульних тестів:
- `RoomServiceTest`
- `BookingServiceTest`
- `CsvGatewayTest`

Тести охоплюють:
- позитивні сценарії;
- невалідні вхідні дані;
- помилки (винятки);
- різні гілки логіки;
- експорт/імпорт із мок-об'єктами (`Writer`).

## Запуск
```bash
mvn compile
mvn test
mvn exec:java -Dexec.mainClass="org.example.hotel.Main"
```

## Висновки
У роботі реалізовано повноцінний консольний застосунок для предметної області "Готель" із чітким розділенням доменної моделі, сервісного шару, вводу/виводу даних та інтерфейсу користувача. Додано юніт-тести для критичних сценаріїв та помилок, а також імпорт/експорт даних із сортуванням. Це дозволило відпрацювати базові навички ООП, проєктування архітектури невеликого застосунку, тестування та роботи з файлами.
